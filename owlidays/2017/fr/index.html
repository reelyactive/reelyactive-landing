La conte Owl-iday 2017

Le cadeau de Timmy.

Le cadeau de Timmy

Le jeune Timmy, qui a reçu une casquette reelyActive pour son anniversaire, a une idée de cadeau pour ses Owl'idays.

Casquette



Sonar

Timmy va au centre commercial, où il voit un nouveau type d'affichage. «Super, je vois quel Elfe a le moins de visiteurs en ce moment!», dit Timmy.

Docs Sonar



BYOD RTLS

Effectivement, la ligne pour rencontrer l'Elfe Chickadee est plus courte! Sonar l'a prédit en comptabilisant anonymement les appareils Bluetooth que les gens portent.

Blogue: BYOD RTLS



Donnée structurée

Timmy partage son voeu avec Chickadee, qui enregistre rapidement les trois éléments importants: Timmy, son voeu et son adresse.

Le format Web standard de Schema.org et JSON-LD garantit que n'importe qui ou n'importe quoi peut donner du sens à cette information.




Base de données

Le voeu de Timmy est immédiatement écrit dans la base de données de l'atelier.

Les elfes n'ont eu aucun problème avec le format de ces nouvelles données. Ils ont grandi avec les technologies Web et ont été inspirés par notre propre json-silo.




Web Dashboard

Le cadeau de Timmy est présenté aux elfes de l'atelier sur une simple page web.

Les elfes n'ont pas non plus eu de problèmes pour le configurer eux-mêmes, en utilisant nos exemples cormorant.js et cuttlefish.js.




Asset Tracking

Une fois fabriqué, les elfes apposent une balise Bluetooth sur le cadeau. Ils associent son ID à la base de données en utilisant notre plate-forme Pareto.

Association Pareto



Visibilité «reel»

L'infrastructure de reelyActive dans l'atelier permet une visibilité et une localisation en temps réel des actifs. Les elfes peuvent tout voir sur Pareto à tout moment, et de n'importe où.

Déploiement Reel



Visibilité éclairée

L'entrepôt lui est équipé d'une infrastructure d'éclairage intelligent flambant neuve qui détecte également les appareils Bluetooth.

Non seulement les elfes trouvent tous leurs paquets, mais ils constatent qu'ils économisent aussi beaucoup sur les coûts de l'énergie!

Blogue: Light hears ahead of its time



Real-Time Location

Récupérer le cadeau de Timmy ne pouvait pas être plus facile. Les elfes ont mis en place un autre tableau de bord en suivant notre tutoriel beaver.js.

Tutoriel beaver.js



Owl-à-bord

Le traîneau est équipé de notre capteur Owl-in-One qui détecte que tous les cadeaux, y compris celui de Timmy, sont à bord.

Le dernier prototype a même un GPS!

Owl-in-One



Live Directory

Avant d'entrer chez Timmy, Mme Owl-iday vérifie d'abord son Live Directory sur Pareto pour s'assurer que tout le monde est au lit.

Docs Live Directory



reelyApp

Mme Owl-iday peut déposer le cadeau. Elle est également visible sur Pareto via son appareil mobile qui exécute notre reelyApp.

Blogue: Advertise yourself with the Physical Web, and beyond...



Owl done!

Le cadeau, déposé au bon endroit, est automatiquement enregistrée. Maintenant, Mme Owl-iday s'envole vers un autre Tims prendre un café pour la longue nuit qui s'annonce!

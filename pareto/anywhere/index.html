Pareto Anywhere

Pioneering context-aware physical spaces

Apply the context (who/what is where/how) of any physical space to any application using the tools you already know and enjoy.

Pareto Anywhere is open source middleware which transforms ambient data into a real-time stream of hyperlocal context which is easily manipulated, searched and analysed using familiar software platforms.

Occupancy analytics
Asset tracking
Personnel tracking
Environmental sensing
Interaction detection

Context for any application

Application-agnostic by design, Pareto Anywhere's web-standard data stream serves any use case—typically many at once.
